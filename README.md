# bp-cognitive-app
MacOS aplikace pro experimentální testování kognitivních funkcí v závislosti na hladině glukózy v krvi. Vyvinuto jako praktická část bakalářské práce na ČVUT Fakultě dopravní. Aplikace umožňuje provádět testy reakčního času, pozornosti a koordinace a ukládat výsledky pro další analýzu.
